# 云原生应用集合

欢迎来到我们的云原生应用集合项目！这里汇集了多个使用现代Web技术构建的单页应用程序，每个应用都专注于解决特定领域的问题。

## 声明

本项目由阿里云ESA提供加速、计算和保护

## 项目概览

### [U-Card - 卡牌收藏管理系统](./U-Card)
一个现代化的卡牌收藏管理平台，允许用户创建、组织和管理各种类型的卡牌收藏。支持多种卡牌类型，提供AI辅助生成功能，以及本地数据存储能力。

### [U-IPTV - 网络电视播放器](./U-IPTV)
一个现代化的网络电视播放平台，允许用户管理和播放各种IPTV频道。支持频道分类管理、播放历史记录、频道搜索功能，以及本地数据存储能力。

### [U-Jobs - 招聘求职管理系统](./U-Jobs)
一个现代化的招聘求职管理平台，允许企业发布招聘信息、求职者投递简历，以及职位管理功能。支持职位分类、在线申请等功能，以及本地数据存储能力。

### [U-Keyboard - 键盘测试工具](./U-Keyboard)
一个现代化的键盘检测和测试平台，允许用户实时检测键盘按键状态、响应速度和连击性能。支持多种键盘布局，提供详细的按键分析功能。

### [U-Note - 轻量级笔记管理系统](./U-Note)
一个现代化的轻量级笔记管理平台，允许用户创建、组织和管理个人笔记。基于浏览器localStorage实现数据存储，无需服务器，完全轻量化设计。

### [U-Tools - 开发者工具箱](./U-Tools)
一个一站式的开发者工具平台，提供多种实用工具，包括JSON格式化、时间戳转换、正则表达式测试、UUID生成、JWT解析、文本对比等。免安装、免注册，即开即用。
