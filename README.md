# 云原生应用集合

欢迎来到我们的云原生应用集合项目！这里汇集了多个使用现代Web技术构建的单页应用程序，每个应用都专注于解决特定领域的问题。所有应用均由阿里云ESA提供加速、计算和保护。

## 声明

## 项目概览

### 📝 [U-MarkdownNote - 浏览器Markdown笔记](./U-MarkdownNote)
功能完善的浏览器端Markdown笔记应用，支持实时预览、语法高亮、笔记管理等功能。无需服务器，所有数据存储在本地，安全可靠。
- **访问地址**: https://u-markdownnote.39b9884f.er.aliyun-esa.net/

### 📱 [U-ResponsiveTest - 响应式设计测试工具](./U-ResponsiveTest)
多设备响应式设计测试工具，支持桌面、平板、手机等多种设备模拟，帮助开发者快速测试页面在不同设备上的显示效果。
- **访问地址**: https://u-responsivetest.39b9884f.er.aliyun-esa.net/

### 🔍 [U-Search - 聚合搜索引擎](./U-Search)
聚合搜索引擎项目，支持多个搜索引擎的聚合搜索功能。用户可以在一个界面中同时搜索多个搜索引擎，并保存搜索历史记录。
- **访问地址**: https://u-search.39b9884f.er.aliyun-esa.net/

### ✅ [U-TaskFlow - 任务流协作管理平台](./U-TaskFlow)
现代化的任务流协作管理平台，帮助团队高效创建、组织和跟踪工作任务与工作流。支持任务板管理、看板视图、工作流设计器等功能。
- **访问地址**: https://u-taskflow.39b9884f.er.aliyun-esa.net/

### 🌤️ [U-WeatherStation - 天气站](./U-WeatherStation)
现代化的天气监测平台，提供实时天气查看、7天天气预报和多城市管理功能。支持自定义单位设置和数据本地存储。
- **访问地址**: https://u-weatherstation.39b9884f.er.aliyun-esa.net/
