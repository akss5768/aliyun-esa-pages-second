# U-DailyTask - 今日任务

U-DailyTask 是一款智能任务管理应用，通过随机生成帮助用户规划每日任务，同时提供任务库管理和历史记录功能。

## 声明

本项目由阿里云ESA提供加速、计算和保护
![阿里云ESA](aliyun.png)

## 功能特性

- **任务编排** - 智能随机生成每日任务清单，按分类和优先级筛选
- **交互功能** - 支持勾选完成的任务管理，实时显示进度
- **自定义设置** - 灵活配置任务类型、优先级和预计时间
- **历史追踪** - 记录用户任务完成情况，查看完成趋势
- **数据持久化** - 使用LocalStorage自动保存所有数据

## 技术栈

- **前端框架**: React 18
- **构建工具**: Vite
- **UI样式**: Tailwind CSS 3.4
- **图标库**: Lucide React
- **日期处理**: date-fns
- **状态管理**: React Hooks
- **数据存储**: LocalStorage

## 使用说明

### 1. 加载或清空数据

- 点击"加载示例数据"可查看预设的任务库
- 点击"清空重置"可清空所有数据，开始创建自己的任务库

### 2. 生成任务

- 选择需要包含的任务分类（工作、学习、健康、生活、其他）
- 筛选优先级（高、中、低）
- 设置最大任务数量
- 点击"生成任务"按钮随机生成今日任务清单
- 点击重置按钮重新配置生成条件

### 3. 今日任务

- 查看生成的每日任务清单
- 点击圆圈勾选完成的任务
- 实时查看完成进度
- 删除不需要的任务
- 查看预计总用时和已完成用时

### 4. 任务库管理

- 添加新任务：输入名称、选择分类、设置优先级和预计时间
- 编辑现有任务：点击编辑按钮修改任务信息
- 删除任务：点击删除按钮移除任务
- 搜索任务：使用搜索功能快速查找任务
- 按分类查看任务

### 5. 历史记录

- 查看每日任务完成情况
- 查看统计数据：记录天数、已完成任务、总任务数、完成率
- 查看完成趋势图表
- 每天完成任务后自动记录到历史

## 任务分类

应用预设以下分类：
- 工作
- 学习
- 健康
- 生活
- 其他

## 优先级说明

- **高优先级**：重要且紧急的任务
- **中优先级**：重要但不紧急的任务
- **低优先级**：不重要且不紧急的任务

## 数据存储

所有数据均保存在浏览器LocalStorage中，刷新页面不会丢失数据。如需清除数据，可：
1. 点击"清空重置"
2. 或在浏览器开发者工具中清除LocalStorage
