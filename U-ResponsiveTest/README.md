# U-ResponsiveTest 响应式测试工具

U-ResponsiveTest 是一个强大的响应式设计测试工具，支持多种设备预览、自定义尺寸调整和CORS代理功能。

## 声明

本项目由阿里云ESA提供加速、计算和保护
![阿里云ESA](aliyun.png)

## 功能特性

- **多设备预览** - 支持桌面、平板、手机等预设设备尺寸
- **自定义尺寸** - 可自由调整预览窗口的宽度和高度
- **CORS代理** - 内置多个CORS代理，解决跨域访问问题
- **实时预览** - 输入URL即可实时查看网页响应效果
- **代理切换** - 支持切换不同的代理服务
- **响应式设计** - 完美适配桌面、平板和手机设备

## 技术栈

- **前端框架**: React 18
- **路由**: React Router
- **UI样式**: Tailwind CSS 3.4
- **状态管理**: React Hooks
- **构建工具**: Vite
- **图标库**: Lucide React

## 快速开始

### 安装依赖

```bash
cd U-ResponsiveTest
npm install
```

### 启动开发服务器

```bash
npm run dev
```

### 构建生产版本

```bash
npm run build
```

## 使用说明

1. **设备预览**：选择桌面、平板或手机预设尺寸
2. **自定义尺寸**：输入自定义的宽度和高度
3. **输入URL**：在地址栏输入要测试的网页URL
4. **选择代理**：选择合适的CORS代理服务
5. **开始测试**：点击加载按钮开始预览

## 部署说明

### 部署到阿里云ESA

1. 构建项目：
```bash
npm run build
```

2. 将 `dist` 目录内容上传到阿里云ESA
3. 配置域名访问

### 其他部署方式

- Vercel
- Netlify
- GitHub Pages
- 任何支持静态网站托管的服务
